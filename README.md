# Helicopter game
I built this game using pygame according to a tutoria in [this book](https://www.empik.com/komputer-swiat-biblioteczka,p1230127088,prasa-p)

## Setup {.tabset}

Prerequisites:
- python >= 3.10

### Windows

```sh
python -m venv .venv
.venv\Source\activate
pip install pygame
```

### Linux / MacOS

```sh
python3 -m venv .venv
source .venv/bin/activate
pip3 install pygame
```
